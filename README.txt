Final Project for CS251: Programming Languages

Interpreter for Scheme language written in C. 

Tokens handled:

boolean, integer, double, string, symbol, parentheses

Primitives handled:

+, car, cdr, null?, cons, -, =, >, <

Also handles:

if, let, let*, letrec, and, or, set!, begin, quote, set!, define, lambda

Example test files given in test-files directory.
